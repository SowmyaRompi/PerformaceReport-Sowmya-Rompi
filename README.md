## Overview
This repository showcases a **Power BI Business Performance Dashboard** designed to analyze
sales, quantity, profit, and year-to-date (YTD) performance across countries, products, and time.

The dashboard demonstrates real-world **business intelligence, data modeling, and DAX skills**
commonly required for Business Data Analyst and Power BI Developer roles.

---

## Key Insights Covered
- Year-to-Date (YTD) vs Previous YTD (PYTD) analysis
- Gross Profit, Sales, and Quantity KPIs
- Country and product-level performance comparison
- Monthly trend analysis
- Interactive filtering and drill-downs
